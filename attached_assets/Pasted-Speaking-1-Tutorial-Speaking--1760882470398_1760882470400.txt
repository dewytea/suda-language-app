Speaking 모듈의 사용성을 개선해줘:

=== 1. 첫 방문 가이드 (Tutorial) ===

Speaking 페이지 처음 진입 시 튜토리얼 표시:

조건 체크:
- localStorage에 'speakingTutorialCompleted' 키가 없으면 튜토리얼 시작
- 있으면 튜토리얼 건너뛰기

튜토리얼 UI:
- 전체 화면 반투명 오버레이 (배경: rgba(0, 0, 0, 0.7))
- 주요 요소만 스포트라이트 (z-index 높게)
- 말풍선 스타일 설명

=== 5. 로딩 상태 개선 ===

5.1 페이지 전환 시
상단에 얇은 진행 바:
- 위치: 화면 최상단 (fixed)
- 높이: 2px
- 색상: #2E5CFF (파란색)
- 애니메이션: 왼쪽에서 오른쪽으로 부드럽게
- 완료 시: Fade out

5.2 음성 분석 중
녹음 버튼 영역에:
┌────────────────────────────┐
│    [로딩 스피너]           │
│                            │
│  AI가 분석 중...           │
│  약 3초 소요              │
└────────────────────────────┘

- 스피너: 원형, 파란색, 회전 애니메이션
- 텍스트: 회색, 16px

5.3 데이터 로딩 중 (통계 페이지)
콘텐츠 영역에 스켈레톤 UI:
- 카드 모양 유지
- 회색 배경
- 펄스 애니메이션 (밝아졌다 어두워졌다)

=== 6. 성공 피드백 개선 ===

6.1 XP 획득 시
현재 위치에 토스트:
┌─────────────┐
│ +20 XP 획득! │
│     ✨      │
└─────────────┘
- 위치: 우측 상단
- 배경: 초록색
- 3초 후 자동 사라짐
- 숫자 증가 애니메이션

6.2 레벨 업 시
전체 화면 모달:
┌────────────────────────────┐
│         🎊                  │
│                            │
│     LEVEL UP!              │
│                            │
│    Level 12 → 13           │
│                            │
│  [폭죽 애니메이션]         │
│                            │
│  [확인]                    │
└────────────────────────────┘
- 배경: 그라데이션 (파란색 → 보라색)
- 애니메이션: 폭죽, 반짝임

=== 7. 사용 흐름 개선 ===

7.1 문장 완료 후
"다음 문장" 버튼 클릭 시:
- 부드러운 전환 애니메이션
- 새 문장 Fade in
- 자동으로 스크롤 상단

7.2 카테고리/난이도 변경 시
- 로딩 없이 즉시 필터링
- 애니메이션: 기존 항목 Fade out, 새 항목 Fade in

7.3 즐겨찾기 토글
- 클릭 즉시 반영 (낙관적 업데이트)
- 하트 애니메이션 (빈 하트 ↔ 꽉찬 하트)
- 성공 시: 그대로 유지
- 실패 시: 원래대로 되돌리고 에러 메시지

=== 8. 접근성 개선 ===

8.1 키보드 네비게이션
- Tab 키로 모든 인터랙티브 요소 접근 가능
- Focus 시 파란색 아웃라인
- Enter/Space로 버튼 클릭

8.2 색상 대비
- 텍스트 대비율 최소 4.5:1
- 버튼 텍스트 읽기 쉽게
